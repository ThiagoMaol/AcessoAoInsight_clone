# Preservação do Acesso ao Insight (Clone de Salvaguarda)

Este repositório é uma iniciativa independente e de salvaguarda do site **[Acesso ao Insight](http://www.acessoaoinsight.net/)**, a maior e mais completa biblioteca do Budismo Theravada em língua portuguesa. Este clone corresponde à **Versão de 31 de Dezembro de 2018**, cujos arquivos originais estão disponíveis publicamente no **[Internet Archive](https://archive.org/details/AcessoAoInsight)**.

O objetivo primordial deste projeto é ajudar a preservar uma sabedoria valiosíssima, viva há milênios, garantindo que o vasto acervo de suttas traduzidos e ensinamentos permaneça sempre acessível à comunidade e aos estudantes do Dhamma, mesmo diante de eventuais instabilidades do site principal.

### 📦 Fonte dos Arquivos

A fonte utilizada para a reconstrução deste clone foi o backup preservado no Internet Archive:
➡️ **[Internet Archive - Acesso ao Insight](https://archive.org/details/AcessoAoInsight)**

> [!IMPORTANT]  
> **Navegue no Site Oficial!**  
> Enquanto o site original estiver ativo e online, este repositório de salvaguarda não faz sentido prático de uso no dia a dia. A melhor experiência de navegação, estudo e uso do conteúdo original e completo está no endereço oficial:  
> ➡️ **[www.acessoaoinsight.net](http://www.acessoaoinsight.net/)**

## 📝 Estado Atual do Repositório

* **Estrutura de Textos:** Este repositório preserva a parte puramente textual (HTML) do site, contendo a íntegra dos suttas e ensaios.
* **Mídias Pesadas:** Para economizar o espaço de armazenamento deste repositório, os arquivos mais pesados de mídia (como audiobooks em `.mp3`/`.m4b`) não foram salvos diretamente nesta hospedagem. Atualmente, o link de tais materiais continua redirecionando para as fontes originais online do site **[www.acessoaoinsight.net](http://www.acessoaoinsight.net/)**.
* **Backup de Salvaguarda Completo:** Todos os arquivos de áudio (audiobooks, podcasts) foram baixados e encontram-se em um backup privado seguro. Caso o site oficial de fato venha a sair do ar de forma definitiva futuramente, esses arquivos de mídia pesada serão disponibilizados em um servidor adequado e este clone (ou outra plataforma semelhante a ser desenvolvida) será ajustado para servi-los de forma direta.
